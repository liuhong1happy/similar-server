# similar-server

## Install

    npm install -g similar-server

## Usage

1. Create template project

        similar-server-cli AwesomeProject

2. Start Server

        cd AwesomeProject
        npm start

4. Open [http://localhost:3002/home/Hello%20Similar%20Server!](http://localhost:3002/home/Hello%20Similar%20Server!)
